# SketchUp 2D Graphic Export Toolbar Button

Adds a toolbar button to trigger a Ruby script that opens the 2D Graphic export dialog. Uses the #send_action method, which is not official API nor is supported. Therefore it is not guaranteed to work in the future, but here's hoping it will for a long time. Pretty sure it only works on Windows.

## Installation
1. Go to **[Releases](../../releases)**.
2. Download the latest `.rbz` file.
3. Open SketchUp.
4. Go to **Extensions > Extension Manager**.
5. Click **Install Extension** at the bottom.
6. Select the downloaded `.rbz` file and open.

## Usage
1. Find a good spot in the toolbar to place the button.
2. When you're ready, press it.
3. The Windows dialog window should open, saving you one (1) click from the usual ***File > Export > 2D Graphic...*** route–unless you have a keyboard shortcut (which I greatly recommend), in which case you save zero (0) clicks.
4. Enjoy!

If, for some reason, you want to export 2D graphics but don't want to use the File tab, this extension is also available in ***Extensions > Export 2D Graphic***. A zero-sum extension... purely for the *looks*.
